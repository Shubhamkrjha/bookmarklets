##  Value Formatter

A useful tool for formatting and transforming text values directly in your browser.

### Features
- **Multiple Output Formats**: SQL IN, SQL VALUES, Python List, JavaScript Array, CSV, Line-separated
- **Case Transformation**: UPPERCASE, lowercase, Title Case
- **Sorting**: A-Z, Z-A, or keep original order
- **Duplicate Detection**: Click on duplicate count to view detailed list with occurrence count
- **Remove Duplicates**: One-click deduplication
- **Trim Whitespace**: Clean up values automatically
- **Value Counter**: Track input/output value counts.


### Use Cases
- Format values for SQL `IN` clauses
- Clean up scraped data
- Remove duplicates from lists
- Convert between different data formats
- Identify and analyze duplicate values

### Installation

1. **Create a new bookmark** in your browser
2. **Name it**: `Value Formatter` (or any name you prefer)
3. **Copy the code** from `value-formatter.js`
4. **Paste it as the bookmark URL**
5. **Save** the bookmark

### Usage

1. Click the bookmarklet on any webpage
2. Paste your values (comma, space, or newline separated)
3. Select output format and options
4. Click **Format**
5. Copy the formatted output

- Does not work on sites with strict Content Security Policy (e.g., Google Docs, Gmail)
- **Workaround**: Open a blank tab (`about:blank`) and run the bookmarklet there
